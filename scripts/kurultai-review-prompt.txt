# Kurultai Review Prompt

You are analyzing Kurultai (6-agent AI system) activity for the past 6 hours.

## CONTEXT

**Agents:**
- Kublai (lead) - Coordination, human interface
- Möngke (research) - Web research, source verification
- Chagatai (content) - Writing, documentation
- Temüjin (dev) - Code, deployment, infrastructure
- Jochi (analysis) - Testing, security, code review
- Ögedei (ops) - Monitoring, health checks, alerts

**LLM Configuration:**
- Local LLM: qwen3.5-9b-mlx (for agent reflections)
- Cloud LLM: qwen3.5-plus (for this analysis)

## DATA SOURCES

You will receive:

1. **CHATLOGS** - Actual conversations between human and Kublai
   - What did the human request?
   - What did Kublai promise to do?
   - What decisions were made?

2. **AGENT REFLECTIONS** - Hourly reflections from each agent
   - What each agent worked on
   - What blockers they encountered
   - What they plan to do next

3. **GIT COMMITS** - What was actually committed
   - Evidence of completed work
   - Files modified

4. **NEO4J ACTIVITY** - Logged decisions and actions
   - KublaiDecision nodes
   - ProcessImprovement nodes
   - Agent status updates

5. **SYSTEM LOGS** - Script execution logs
   - What scripts ran
   - Success/failure status

## ANALYSIS REQUIRED

### 1. What Worked Well
List specific evidence from the data. Cite specific:
- Chatlog messages
- Git commits
- Neo4j entries
- Log entries

### 2. What Didn't Work
List specific evidence. Be specific about:
- What failed
- Where in the data it's visible
- Impact on goals

### 3. Patterns Across Agents
Identify recurring themes:
- Are multiple agents blocked on the same thing?
- Are there communication gaps?
- Are there efficiency opportunities?

### 4. Kublai Performance Review
Critical analysis:
- Did Kublai complete what was promised to the human?
- What's blocking progress?
- Response time to human requests
- Quality of follow-through

### 5. Recommended Process Improvements
Specific, actionable improvements:
- What should change
- Why it will help
- How to implement

### 6. Priority Action Items (Next Hour)
Top 3-5 actions Kublai should take immediately:
- Must be actionable
- Must be specific
- Must be prioritized

## QUALITY CRITERIA

Your analysis must be:

1. **Evidence-Based** - Every claim cites specific data
2. **Specific** - No vague statements
3. **Actionable** - Recommendations can be executed
4. **Honest** - Call out failures, not just successes
5. **Prioritized** - Most important items first

