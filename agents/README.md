# Kurultai installer agent pack

This directory contains prompts/contracts for agents installing and configuring Kurultai from this repository.

- `hermes-install-expert.md` — the expert operating prompt for a local coding agent. Use it together with `docs/operations/fresh-install-agent-prompt.md` and `docs/operations/full-installation-checklist.md`.

The install expert is intentionally public and non-secret. It teaches process, verification, and safety boundaries; it does not contain live credentials, private chat IDs, OAuth tokens, sessions, or private Brain contents.
