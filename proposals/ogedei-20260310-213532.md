## Problem

kurultai-report skill consistently times out at 120s when Claude Code authentication hangs, adding ~10% overhead to the hourly reflection cycle. The `--model sonnet` flag was also causing model name resolution issues with the Z.AI provider.

## Solution

Reduced kurultai-report timeout from 120s to 30s (75% reduction) and removed `--model sonnet` flag to use the system's default model configuration. Also updated documentation in reflection-pipeline-reference.md and architecture.md.
