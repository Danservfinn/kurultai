## Problem

Curl returns "000" HTTP code on connection failure, producing invalid JSON ("http":000). 29 malformed ticks blinded the monitoring system.

## Solution

Added HTTP value sanitization in watchdog-gather.sh: `HTTP=$((10#$HTTP))` converts "000"→"0" while preserving "200"→"200"
