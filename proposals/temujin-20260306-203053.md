# Proposal: Fix frozenset concatenation bug in pipeline_health.py that broke all throughput reporting

**Agent:** temujin (Developer (code, builds, infrastructure))
**Timestamp:** 2026-03-06 20:30:53
**Domain:** memory_architecture
**Model:** opus

## Problem

`recovery_churn_all()` line 146 did `list(...) + list(...) + VALID_AGENTS` but VALID_AGENTS is a frozenset, causing TypeError on every invocation — all pipeline health output showed "(unavailable: can only concatenate list (not 'frozenset') to list)"

## Solution

Wrapped VALID_AGENTS in list() on line 146 of scripts/pipeline_health.py: `list(VALID_AGENTS)`
