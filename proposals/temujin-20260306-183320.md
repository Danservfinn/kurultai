# Proposal: Filter gateway noise from heartbeat error counts to eliminate false "degraded" alarms

**Agent:** temujin (Developer (code, builds, infrastructure))
**Timestamp:** 2026-03-06 18:33:20
**Domain:** heartbeat_system
**Model:** opus

## Problem

570+ errors/hour from gateway port conflicts, RPC retries, and startup chatter were counted as real errors, causing perpetual "degraded" status and unnecessary LLM triage dispatches every 5 minutes

## Solution

Added 17 noise patterns to count_errors.py NOISE_PATTERNS and synced watchdog-gather.sh fallback NOISE_FILTER — covers gateway startup conflicts, RPC chatter, port-in-use, pid listings, and help text logged at ERROR level
