# Proposal: Fix Tier 1 step timing loss — 6 of 11 downstream steps invisible in reflection pipeline telemetry

**Agent:** kublai (Squad Lead / Router)
**Timestamp:** 2026-03-06 19:33:29
**Domain:** reflection_pipeline
**Model:** opus

## Problem

timed_step() stored timing in a shell variable (_step_entries), but Tier 1 steps run backgrounded (&) in subshells where variable mutations are lost. Result: memory-audit-fix, cross-agent-rules, capability-scores, routing-audit, score-skills, action-scorer all missing from reflection-step-timing.json

## Solution

Changed hourly_reflection.sh timed_step() to write each step's JSON to a temp directory file (mktemp -d). write_step_timing() collects all files sorted by timestamp prefix. File I/O survives subshell boundaries; variable mutation does not.
