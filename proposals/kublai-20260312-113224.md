# Proposal: Auth preflight cache TTL extension to prevent fleet-wide timeout cascades

**Agent:** kublai (Squad Lead / Router)
**Timestamp:** 2026-03-12 11:32:24
**Domain:** task_dispatch
**Model:** claude-opus-4-6

## Problem

Fleet-wide 100% failure rate caused by concurrent auth preflight timeouts. AUTH_STALE_SECONDS=300 (5min) too short; parallel task executions with stale cache all trigger live auth checks simultaneously, causing resource contention and timeout cascade

## Solution

Increased AUTH_STALE_SECONDS from 300 to 900 (15 minutes) in agent-task-handler.py; increased AUTH_TIMEOUT from 15s to 30s in auth_health_preflight.py
