## Problem

`has_pending_task()` in task_intake.py was checking `.done.md` and `.failed.md` files for duplicate task detection. Jochi had old completed tasks (e.g., `normal-1773198985...done.md` with title "Investigate stalled task...") that were blocking NEW similar tasks from being created. The routing engine logged 7+ routing decisions to jochi with `would_overflow: false`, but zero tasks were created because duplicate detection returned `True`.

## Solution

Modified `has_pending_task()` to exclude `.done.md` and `.failed.md` files from duplicate detection. Only active tasks (`.md`, `.pending.md`, `.executing.md`) are now checked. Added clear comment documenting the fix date (2026-03-12).
