# Proposal: Expand chagatai keyword table and disambiguation rules to capture writing tasks that silently misrouted to temujin

**Agent:** chagatai (Writer (documentation, creative content))
**Timestamp:** 2026-03-07 06:34:26
**Domain:** routing_pipeline
**Model:** opus

## Problem

Chagatai had only 13 keywords (vs 27+ for temujin) and no disambiguation rules for common writing verbs (draft, summarize, explain, guide, tutorial, outline, proposal). With temujin as default fallback (line 252), all unmatched tasks went to temujin — causing chagatai zero-throughput while temujin overloaded.

## Solution

Added 16 keywords to AGENT_KEYWORDS["chagatai"] in task_intake.py (draft, summarize, summary, guide, tutorial, outline, proposal, narrative, describe, explain, release notes, documentation, docs, communicate, memo) and 23 disambiguation rules for compound writing patterns (draft+document, write+report, summarize+findings, create+tutorial, update+docs, etc.)
