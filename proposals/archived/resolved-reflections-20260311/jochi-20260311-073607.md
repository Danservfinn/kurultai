## Problem

watchdog-gather.sh was writing malformed JSON to ticks.jsonl — JSON template had 4 service placeholders but argument list passed 8 variables, causing variable shift that produced values like `"pending":"healthy"` instead of numbers. Also, pgrep returned multiple PIDs with embedded newlines breaking JSON structure.

## Solution

(1) Added missing cloudflared fields to JSON template: `"services":{"neo4j","redis","cloudflared","cloudflared_pid","recovery":{"neo4j"...,"cloudflared"...}}` (2) Sanitized CLOUDFLARED_PID with `tr '\n' ',' | sed 's/,$//'` to replace newlines with commas
