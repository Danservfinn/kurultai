# Parse Content Generation Workflow

## 🎯 Strategy

1. **I analyze** news articles using Parse (autonomous)
2. **I generate** X/Twitter threads + Reddit posts based on results
3. **You copy/paste** and post from your accounts
4. **I monitor** engagement, iterate on what works

---

## 📋 Workflow

### Daily Content Generation

```
Morning (7 AM EST):
  1. I select 3-5 trending news articles
  2. I analyze each with Parse
  3. I generate content packages:
     - X/Twitter thread (5-7 tweets)
     - Reddit post (r/journalism, r/MediaCriticism)
     - LinkedIn post (optional)
  4. I deliver to you in copy/paste format
  5. You post when convenient

Evening (7 PM EST):
  1. I analyze engagement metrics
  2. I identify top-performing content
  3. I iterate on successful formats
  4. I report what's working
```

---

## 📝 Content Templates

### X/Twitter Thread Format

```
Tweet 1/7: Hook + Parse analysis result
Tweet 2/7: Key finding #1 (truth score)
Tweet 3/7: Key finding #2 (propaganda techniques)
Tweet 4/7: Key finding #3 (bias detection)
Tweet 5/7: What this means for readers
Tweet 6/7: Call to action (try Parse)
Tweet 7/7: Link to Parse
```

### Reddit Post Format

```
Title: [Analysis] Article about X shows [interesting finding]

Body:
- Context (why this matters)
- Methodology (Parse 8-agent AI analysis)
- Key findings (bullet points)
- Full analysis link (Parse)
- Question for community (engagement)
```
