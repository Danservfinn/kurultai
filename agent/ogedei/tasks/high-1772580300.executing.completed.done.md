# Task: Fix heartbeat-watchdog Cron Configuration Error

## Issue Identified

The heartbeat-watchdog cron job has incorrect paths:

**Current (WRONG):**
```bash

```

**Problems:**
1. Script path `/Users/kublai/scripts/heartbeat_monitor.sh` doesn't exist
2. Actual script is at `/Users/kublai/.openclaw/agents/main/scripts/heartbeat-watchdog.sh`
3. Log file `/tmp/heartbeat_monitor.log` should be in proper logs directory

